Suggested Management Plan

Assessment: Septic shock (urosepsis, ESBL source) with AKI Stage 3 and early ARDS.

1. Source control & antibiotics: Meropenem 1g IV q12h (renal-adjusted for eGFR). Send repeat cultures.
2. Haemodynamics: Target MAP ≥ 65 mmHg. Uptitrate noradrenaline; add vasopressin 0.03 U/min if >0.25 mcg/kg/min. Consider hydrocortisone 200 mg/day if refractory.
3. Renal: AKI 3 with oliguria — initiate CRRT (CVVHDF, 25 ml/kg/hr).
4. Respiratory: Lung-protective ventilation (Vt 6 ml/kg PBW), monitor for proning if P/F < 150.
5. Monitoring: Lactate clearance, hourly urine output, daily cultures.
